# 🚀 GESI AI PREMIUM V4.0 (Local Edition)

A high-performance, local-first AI Forex trading bot integrated with **M-Pesa** for autonomous deposits and hyper-growth trading. No cloud, no ngrok, 100% private.

## 💎 Premium Features (V4.0)
- **Zero-Latency M-Pesa Integration**: Fully automated STK Push for deposits and B2C for withdrawals. Works instantly from your local machine.
- **AI Moonshot Pro Strategy**: Enhanced LSTM trend prediction with a 99.8% entry confidence requirement.
- **Local Dashboard hosting**: No more Vercel 404 errors. The Python engine hosts the dashboard on your machine.
- **Zero-Loss Protection**: Dynamic trailing stops and ATR-based volatility protection.

## 🛠️ How to Start Trading

### 1. The Launcher (One-Click)
Double-click the **`start_local.bat`** file in the root directory. This will:
1. Automatically install any missing dependencies.
2. Start the GESI AI Trading Engine.
3. Host your Premium Dashboard.

### 2. Access the Dashboard
Once the engine is running, open your web browser and go to:
👉 **[http://localhost:5000](http://localhost:5000)**

## ⚙️ Configuration
Make sure your **`.env`** file is correctly set with your MT5 and M-Pesa credentials.
